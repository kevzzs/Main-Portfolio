My Works

Plane Essential

This project is a frontend web application created as practice for building a real-world clothing brand website. It features a thrift clothing concept where users can browse items such as shirts, shorts, and jackets through a clean and responsive interface.

Manhwa Reader

This project is a manhwa reading website built to create a community-driven platform for manhwa enthusiasts. Users can explore different manhwa series, search by genre, and upload their own content, while also engaging with other readers through a built-in chat feature. The goal of the project is to combine content consumption and social interaction in one accessible and user-friendly website.

Digital Birthday Gift

A fully interactive digital birthday invitation website that showcases the celebrant, event countdown (days, hours, minutes, and seconds), venue details, FAQs, color-themed design elements, and a gallery section for prenup and celebratory photos. The project combines responsive layouts, visual aesthetics, and user-friendly navigation to create an engaging online invitation experience.

Clothing Showcase

A responsive clothing showcase website that displays a variety of apparel including jackets, shorts, shirts, jerseys, and more. Each product includes detailed information such as available sizes, allowing users to easily browse and explore the collection in an organized and visually appealing layout.
